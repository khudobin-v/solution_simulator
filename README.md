# Симулятор растворения

Приложение для моделирования растворения твёрдых веществ методом клеточного автомата.  
Состоит из двух частей: **сервер** (FastAPI) и **клиент** (Flutter).

---

## Требования

| Компонент | Версия |
|-----------|--------|
| Python    | 3.10+  |
| Flutter   | 3.x (Dart SDK ≥ 3.12) |

---

## Запуск сервера

```bash
cd server
bash start.sh
```

При первом запуске скрипт автоматически:
1. Создаёт виртуальное окружение `.venv`
2. Устанавливает зависимости из `requirements.txt`
3. Запускает сервер на `http://localhost:8000`

Сервер работает в режиме `--reload` — изменения в коде применяются без перезапуска.

---

## Запуск клиента

```bash
cd client
flutter pub get
flutter run
```

Flutter предложит выбрать платформу (macOS, Chrome, и т.д.).  
Для запуска только на macOS:

```bash
flutter run -d macos
```

Клиент подключается к серверу по адресу `http://localhost:8000` — сервер должен быть запущен до старта клиента.

## Первый вход

При первом открытии приложения нужно зарегистрировать аккаунт — данные хранятся локально в `server/app.db` (SQLite).
